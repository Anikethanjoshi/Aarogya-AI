## 🚀 Quick Start

### Prerequisites
- Node.js 18+ and npm
- Modern web browser
- Internet connection for API services

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Anikethanjoshi/aarogya-ai.git
   cd aarogya-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   ```
   
   Edit `.env` with your API keys:
   ```env
   VITE_TAVUS_API_KEY=your_tavus_api_key_here
   VITE_REVENUECAT_API_KEY=your_revenuecat_api_key_here
   VITE_GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Navigate to `http://localhost:5173`

## 🔧 Configuration

### Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `VITE_TAVUS_API_KEY` | Tavus AI API key for video agents | Yes |
| `VITE_REVENUECAT_API_KEY` | RevenueCat API key for subscriptions | Yes |
| `VITE_GOOGLE_MAPS_API_KEY` | Google Maps API key for location services | Optional |
| `VITE_API_BASE_URL` | Backend API base URL | Optional |

### Tavus AI Setup

1. Sign up at [Tavus.io](https://tavus.io)
2. Create a new project
3. Get your API key from the dashboard
4. Add it to your `.env` file

### RevenueCat Setup

1. Sign up at [RevenueCat](https://revenuecat.com)
2. Create a new app
3. Configure your products and packages
4. Get your API key and add to `.env`
